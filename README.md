## Kjør eller bruk prosjektet

Sjekk det ut online på: https://bysykkel.vercel.app/

---

kjør lokalt med å: 

1. Installer PNPM (https://pnpm.io/installation)
2. Naviger til root og run "pnpm install"
3. run "pnpm run dev"
4. go to localhost:5173 i browser
